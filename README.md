# Arrays in Java

This folder contains Java programs that demonstrate the core concepts of arrays.
The programs are written in a simple and beginner-friendly manner.

## Topics Covered
- Array declaration and initialization
- Array input and output
- Finding maximum and minimum elements
- Reversing an array
- Searching elements in an array
- Sorting arrays (basic sorting techniques)
- 2D arrays

## Purpose
This folder is created for practicing and revising array concepts in Java,
which are fundamental for Data Structures and Algorithms.

## How to Use
- Open any .java file
- Run the program using any Java IDE (IntelliJ IDEA recommended)
- Modify inputs to practice more
